# 하우스키핑 v3 — Claude 행동 규칙

## 1. 코드 구현 원칙
- **구현하라고 할 때까지 코드를 작성하지 않는다**
- 설계, 계획, 구조 설명은 자유롭게 하되 실제 코드는 명시적 지시 후에만 작성

## 2. 코드 주석
- 모든 주석은 **한글**로 작성
- 로직이 자명하지 않은 경우 반드시 한글 설명 추가

## 3. 질문 답변 방식
- 사용자가 방법을 물어보면 **선택지 4가지**를 제시
- 각 선택지에 아래 표시 중 하나를 붙임:
  - `⭐ 베스트` — 이 상황에 가장 적합한 방법
  - `🔶 중간` — 괜찮지만 트레이드오프 있음
  - (표시 없음) — 가능하나 비추천

## 4. Supabase 무료 플랜 준수
- 목표: **Supabase Free Plan 내에서만 동작**하는 앱
- 무료 한도:
  - Storage 1 GB / Bandwidth 10 GB/월 / DB 500 MB / MAU 50,000
- 과금 유발 가능성이 있는 구현이면 **즉시 경고** 후 대안 제시

## 5. 대화 종료 처리
- 사용자가 **"대화종료"** 라고 말하면:
  1. 설계서(`housekeeping_v3_설계서_무료버전.md`) 변경사항 업데이트
  2. 다음 세션에서 이어할 내용을 메모리에 저장
